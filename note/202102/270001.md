# 栈

<script type="text/javascript" src="/include/head.js"></script>

栈顶指针为top，指向栈顶元素

```
int stack[n], top = -1;
```

向栈顶推入一个数x

```
stack[++top] = x;
```

从栈顶弹出一个数

```
top--;
```

获取栈顶的数
```
stack[top];
```

注解：

(1)

出栈、取栈顶需判断栈不空

```
top != -1 // 栈不空的条件
```

入栈需判断栈不满

```
top != n-1 // 栈不满的条件
```
